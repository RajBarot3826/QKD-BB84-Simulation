# BB84 Quantum Key Distribution Simulation

This project implements a complete end-to-end simulation of the BB84 Quantum Key Distribution (QKD) protocol.

## Features

- Raw Key Generation
- Basis Sifting
- Eavesdropping Detection (QBER)
- Error Correction
- Privacy Amplification
- Final Secure Key Generation

## Files

- main.py → Main execution file
- alice.py → Alice quantum sender
- bob.py → Bob quantum receiver
- channel.py → Quantum channel simulation
- error_correction.py → Error correction module
- privacy_amplification.py → Privacy amplification module

```bash
python main.py
